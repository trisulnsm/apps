# ISP Path Analytics

Install this app to get

 - Visualization of BGP Routes 
 - Route queries per Peer, Peer Interface, or Network wide
 - Breakup of path segments 
 - Hop by hop analysis
 - Sankey views
 - Ability to filter , exclude dominating paths , export to XLSX
